# 隐私政策

**最后更新：2024 年**

## 数据收集

NotiMon **不会收集**任何个人信息。所有数据处理均在设备本地完成。

## 数据权限

- **通知访问权限** — 仅用于读取通知内容，用于转发
- **短信权限** — 仅用于读取收到的短信，用于转发
- **网络权限** — 仅用于向用户配置的 Webhook / SMTP 推送消息

## 数据存储

配置规则存储在本地 SharedPreferences 中，不会上传到任何服务器。

## 第三方服务

本应用不使用任何第三方分析或广告 SDK。

## 联系我们

如有隐私相关问题，请在 GitHub 提交 Issue。
